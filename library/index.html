Task:

Rəqəmsal Kitabxana Sistemi
Bu tapşırıqda sizdən bir kitabxana idarəetmə sistemi qurmaq tələb olunur. Eyni məntiqi iki fərqli yanaşma ilə reallaşdırmalısınız: Constructor Function və ES6 Class.

Tapşırığın Şərtləri:
Hər iki variantda yaradılan obyektlər aşağıdakı xüsusiyyətlərə və metodlara malik olmalıdır:

Properties (Xüsusiyyətlər):
title: Kitabın adı.
author: Müəllifin adı.
isAvailable: Kitabın kitabxanada olub-olmadığını bildirən boolean dəyər (ilkin dəyər: true).
dueDate: Kitabın qaytarılmalı olduğu tarix (ilkin dəyər: null).

Methods (Metodlar):
borrowBook():
Əgər kitab kitabxanadadırsa (isAvailable true), onu götürülmüş hesab etməli (false etməli) və dueDate cari tarixdən 14 gün sonranı təyin etməlidir.
Əgər kitab artıq götürülübsə, konsola müvafiq xəbərdarlıq çıxarmalıdır.

returnBook():
Kitabın statusunu yenidən əlçatan (true) etməli və dueDate dəyərini null olaraq sıfırlamalıdır.

isOverdue():
Kitabın qaytarılma tarixinin keçib-keçmədiyini yoxlamalıdır. Cari tarixlə dueDate-i müqayisə edərək true və ya false qaytarmalıdır.

Tələblər:
Variant A (Constructor Function): Obyekt strukturunu funksiya daxilində qurun, lakin yaddaş effektivliyi üçün metodları mütləq .prototype üzərindən əlavə edin.
Variant B (ES6 Class): Eyni funksionallığı modern class sintaksisindən və constructor metodundan istifadə edərək yazın.

Qeyd: Kodunuzun işləməsini yoxlamaq üçün ən azı 2 fərqli kitab obyekti yaradın və metodları çağıraraq nəticələri konsolda izləyin.
