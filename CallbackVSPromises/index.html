Callback vs Promises

Tasks

Bir kafe üçün proqram yazırsınız. Müştəri qəhvə sifariş verir.

sifarisVer(miqdar) adlı bir funksiya yaradın. Bu funksiya yeni bir Promise qaytarmalıdır.

Qəhvənin hazırlanması 3 saniyə çəkir (setTimeout istifadə edin).

Məntiq:

Əgər miqdar 5-dən çoxdursa, Promise reject olunsun: "Bağışlayın, eyni anda maksimum 5 qəhvə hazırlaya bilərik."

Əgər miqdar 0-dırsa, Promise reject olunsun: "Zəhmət olmasa miqdar qeyd edin."

Əgər hər şey qaydasındadırsa, 3 saniyə sonra resolve olunsun: "${miqdar} ədəd qəhvəniz hazırdır!"

Tələb: Bu funksiyanı həm .then()/.catch(), həm də async/await ilə çağırıb nəticəni konsolda göstərin.

function sifarisVer(yemek, callback) {
console.log(${yemek} sifariş qəbul edildi...);
setTimeout(() => {
callback(yemek);
}, 2000); }

function hazirla(yemek, callback) {
console.log(${yemek} hazırlanır...);
setTimeout(() => {
callback(${yemek} hazırdır!);
}, 3000); }

Callback hell ile funcksiyani call et. Daha sonra promise cevirib then ve catch istifade ederek call et.
