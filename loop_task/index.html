const cashier = { calculateTotal: function(tax, discount) { const total = this.price + (this.price * tax) - discount; console.log(${this.customerName} üçün yekun məbləğ: ${total} AZN); } };

const customer1 = { customerName: "Aysel", price: 100 };

Tapşırıq Şərtləri:

Object.keys() istifadə edərək bütün vəzifələrin (key) adını böyük hərflərlə konsola çıxarın.

for...in dövrü vasitəsilə maaşı 3000-dən yuxarı olan işçilərin siyahısını ("Vəzifə: Maaş" formatında) çap edin.
